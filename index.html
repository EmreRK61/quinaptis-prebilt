IM/WM Demo App

Overview

Welcome to the PreBilt™ for IM/WM demo application, which significantly improves the user experience for warehouse operators using IM/WM. Built with the user in mind, the solution dramatically increases productivity, improves stock accuracy and reduces training times, through enhanced functionality that will be shown in the demo.

PreBilt for IM/WM can be deployed onto an SAP system in days, with the flexible configuration framework allowing screen changes to be made in minutes, such as hiding fields, changing positions, increasing font size or changing font colours.

Selection Screen

Overview

Welcome to Goods Receipt External, this function allows the user to receive stock into a warehouse, manufacturing plant or distribution centre operating on SAP Inventory Management or Warehouse Management. Goods Receipt External can work with different builds of SAP, this demo showcases just one example. The purpose of the selection screen is to reduce the number of documents the app returns based on selection criteria. The user can enter a specific PO number (4500000000 or 4500000405), vendor (VENDOR1 and VENDOR2), date range, a combination of these fields or they can leave all fields blank to return all possible purchase orders.

Press ‘Select’ to progress through the app.

Press the back arrow to exit the application.

Business Scenario

  • A purchase order is raised and placed on the vendor.
  • The physical goods arrive on site.

Goods Receipt Function

  • The user selects the relevant purchase order.
  • The purchase order header information is reviewed.
  • The user has the option to capture images and signatures for evidence and audit purposes.
  • Each item is processed and verified.
  • The goods receipt is posted back to SAP.
Input Item

Overview

The user enters the quantity of the material to be received and selects a storage location to receive the stock into. The Batch Value field can be left blank, to generate an internal batch number, or a batch number can be entered. The app will determine if the material requires any batch information and prompt the user to enter the details.

Select ‘Capture Image or Signature’ to capture images and signatures to attach to the purchase order.

The chosen item will be added to the basket upon pressing save.

Document Header

Overview

On this screen, the user can view information on the purchase order header.

Select ‘Capture Image or Signature’ to capture images and signatures to attach to the purchase order.

Press ‘Continue’ to progress through the app.

Purchase Order List

Overview

This screen displays a list of purchase orders that fulfil the selection criteria. The user selects a document from the list to view the document header information.

In the header summary there is a count of the number of documents in the list.

The purchase order field is an example of how a field in PreBilt can be configured to change the font colour, size and style.

The filter bar can be used to reduce the number of documents on the list. The user can filter by purchase order number, vendor or vendor name.

Press the back arrow to return to the previous screen.

Image Capture

Overview

Press ‘Upload Image’ to upload an existing image on your device or to launch the camera to take a picture. Once an image has been selected / captured, it will be added to the list and can be selected or deleted. The header displays the maximum and minimum number of images that can be captured and the total number already uploaded.

If the minimum number of images configured is 0 the ‘Skip’ button will be available.

Press ‘Upload’ to attach the images in the list to the Purchase Order.

Basket

Overview

This list displays all items previously added to the basket that can be posted for goods receipt against the chosen purchase order.

Press ‘Add New Item’ to add another item to the basket.

Press ‘Empty Basket’ to remove all items from the basket.

Upon selecting ‘Post’ the contents of the basket will be submitted for goods receipt against the document selected.

Signature Capture

Overview

Press ‘Add Signature’ to add a signature and signee name. Once a signature has been captured it will be added to the list and can be selected or deleted. The header displays the maximum and minimum number of signatures to be captured, along with the total number already uploaded.

If the minimum number of signatures configured is 0 the ‘Skip’ button will be available.

Press ‘Upload’ to attach the signatures in the list to the Purchase Order.

Attachment List

Overview

This screen displays any existing attachments for the document selected. To begin capturing attachments, the user selects ‘Continue’ to go to the first utility automatically or uses the kebab menu to manually select a utility.

Press ‘Capture Signature’ to capture a signature as an attachment on the Purchase Order.

Press ‘Upload Image’ to attach an existing image on the device to the Purchase Order.

Press the back arrow to return to the previous screen.
